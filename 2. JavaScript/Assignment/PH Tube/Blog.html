Scope of var, let, and constVar :

Var :

It's have Global and Functional scope . In the case of Global scope we can access and modify it any where in the Program.

In The case of Functional scope we can access and modify it any where into the function only

Let :

It's a Block Scope variable type .A block is a part of code that surrounded by curly braces {}. We can access it into the block and also we can modify it .

Const :

It's Scope also block like let . we can only access it into the block but It's immutable we can not modify it after declaration

NULL :

It's a empty value that is intentional . If we don't have any meaningful Data there we use NULL

Undefined :

variable has been declared, but its value has not been assigned there we can call it undefined. It represents the lack of initialization.
